Real Numbers & Expressions

The Real Number System

  • Natural numbers: 1, 2, 3, ... (counting numbers)
  • Whole numbers: 0, 1, 2, 3, ...
  • Integers: ..., -2, -1, 0, 1, 2, ...
  • Rational numbers: any number that can be written as a fraction p/q where q ≠ 0 (includes terminating and repeating decimals)
  • Irrational numbers: cannot be written as a fraction (non-terminating, non-repeating decimals). Examples: √2, π, e
  • Real numbers: all rational and irrational numbers together

Exponent Rules

Radicals & Rational Exponents

  • a1/n = √a (the nth root of a)
  • am/n = (√a)m = √am
  • To simplify radicals: factor out perfect squares (or cubes, etc.)
  • Rationalizing the denominator: multiply by a form of 1 to eliminate radicals from the denominator
  • You cannot take the even root of a negative number in the real number system

Factoring Essentials

  • GCF first: always factor out the greatest common factor first
  • Difference of squares: a² - b² = (a+b)(a-b)
  • Perfect square trinomials: a² + 2ab + b² = (a+b)²
  • Sum/difference of cubes: a³ + b³ = (a+b)(a² - ab + b²) and a³ - b³ = (a-b)(a² + ab + b²)
  • Trinomial (ac method): for ax² + bx + c, find two numbers that multiply to ac and add to b
"SOAP" for sum/difference of cubes: Same sign, Opposite sign, Always Positive.

Solving Equations

Linear Equations

  • Form: ax + b = c
  • Solve by isolating the variable: use inverse operations
  • One solution, no solution (contradiction), or infinitely many solutions (identity)
  • Clear fractions by multiplying all terms by the LCD

Quadratic Equations

Standard form: ax² + bx + c = 0

  • Factoring: set each factor equal to zero
  • Square root method: if ax² + c = 0 or (x + k)² = d
  • Completing the square: rewrite in the form (x + h)² = k
  • Quadratic formula: x = (-b ± √(b² - 4ac)) / 2a

The Discriminant

Δ = b² - 4ac

Discriminant ValueNumber & Type of Solutions
Δ > 0 (perfect square)Two distinct real, rational solutions
Δ > 0 (not perfect square)Two distinct real, irrational solutions
Δ = 0One repeated real solution (double root)
Δ < 0No real solutions (two complex conjugates)

Other Equation Types

  • Absolute value: |ax + b| = c means ax + b = c OR ax + b = -c (c must be ≥ 0)
  • Radical equations: isolate the radical, raise both sides to the appropriate power, always check for extraneous solutions
  • Rational equations: multiply by the LCD, solve, check that solutions don't make any denominator zero
  • Equations in quadratic form: substitute u for a repeated expression (e.g., x&sup4; - 5x² + 4 = 0: let u = x²)

Inequalities

Linear Inequalities

  • Solve like a linear equation, but flip the inequality sign when multiplying or dividing by a negative number
  • Express solutions using interval notation or a number line
  • Compound inequalities: "and" = intersection (∩), "or" = union (∪)

Interval Notation

InequalityIntervalBoundary
a < x < b(a, b)Open (excluded)
a ≤ x ≤ b[a, b]Closed (included)
x > a(a, ∞)Always open at ∞
x ≤ b(-∞, b]Always open at -∞
Parentheses = open circle on number line (not included). Brackets = closed circle (included). Infinity always gets a parenthesis.

Absolute Value Inequalities

  • |x| < a means -a < x < a (AND/between: the solution is an interval)
  • |x| > a means x < -a OR x > a (OR/outside: the solution is two rays)
  • "Less than" = "and" (compound). "Greater than" = "or" (two pieces).
"Less thAND" and "GreatOR"

Polynomial & Rational Inequalities

  • Get zero on one side, factor completely
  • Find critical values (zeros and undefined points)
  • Use a sign chart (test intervals) or number line to determine where the expression is positive or negative
  • For rational inequalities: never multiply both sides by an expression containing the variable (sign is unknown)
  • Write the answer in interval notation

Functions: Definitions & Properties

What Is a Function?

  • A relation where each input (x) maps to exactly one output (y)
  • Vertical Line Test: a graph is a function if no vertical line crosses it more than once
  • Notation: f(x) means "the output of function f at input x"
  • f(x) is NOT f times x. It is function notation.

Domain & Range

  • Domain: all possible input (x) values
  • Range: all possible output (y) values
  • Common domain restrictions: cannot divide by zero, cannot take even root of a negative, cannot take log of zero or negative
  • For f(x) = 1/(x-3), domain is all real numbers except x = 3
  • For f(x) = √(x-2), domain is x ≥ 2, or [2, ∞)

Evaluating & Interpreting Functions

  • To evaluate f(a): substitute a for every x in the expression
  • Difference quotient: [f(x+h) - f(x)] / h. This measures the average rate of change and is the foundation of calculus.
  • Finding where f(x) = 0 means finding the x-intercepts (zeros/roots)
  • Finding f(0) gives the y-intercept

Increasing, Decreasing, Constant

  • A function is increasing on an interval if f(x) goes up as x moves right
  • A function is decreasing on an interval if f(x) goes down as x moves right
  • A function is constant on an interval if f(x) stays the same
  • State increasing/decreasing intervals using x-values (not y-values), with open parentheses at turning points

Even & Odd Functions

  • Even function: f(-x) = f(x) for all x. Graph is symmetric about the y-axis. Examples: x², |x|, cos(x).
  • Odd function: f(-x) = -f(x) for all x. Graph is symmetric about the origin. Examples: x³, x, sin(x).
  • Most functions are neither even nor odd.

Piecewise Functions

  • Defined by different rules on different intervals of the domain
  • Evaluate by first checking which interval x falls in, then using the corresponding rule
  • Watch for open vs. closed circles at boundary points when graphing
  • Common example: absolute value as a piecewise function: |x| = x if x ≥ 0, and -x if x < 0

Transformation Rules

Starting from y = f(x):

  • f(x) + k: shift up k units
  • f(x) - k: shift down k units
  • f(x - h): shift right h units
  • f(x + h): shift left h units
  • -f(x): reflect over the x-axis
  • f(-x): reflect over the y-axis
  • a·f(x): vertical stretch (a > 1) or compress (0 < a < 1)
  • f(bx): horizontal compress (b > 1) or stretch (0 < b < 1)
Horizontal shifts are counterintuitive: f(x - 3) shifts RIGHT 3. Think "opposite of the sign inside."

Order of Transformations

  • Apply transformations in this order: horizontal shifts and stretches first (inside the function), then vertical stretches and reflections, then vertical shifts (outside the function)
  • When reading y = a·f(b(x - h)) + k, the vertex/anchor point moves to (h, k), and a and b control stretch/compression

Operations on Functions

Arithmetic Combinations

  • (f + g)(x) = f(x) + g(x)
  • (f - g)(x) = f(x) - g(x)
  • (f · g)(x) = f(x) · g(x)
  • (f / g)(x) = f(x) / g(x), where g(x) ≠ 0
  • The domain of the combined function is the intersection of the domains (with additional restrictions for division)

Composition of Functions

  • (f ∘ g)(x) = f(g(x)): "f of g of x"
  • First evaluate the inner function g(x), then plug the result into f
  • Order matters: f(g(x)) ≠ g(f(x)) in general
  • Domain of f(g(x)): all x in the domain of g such that g(x) is in the domain of f

Inverse Functions

  • f and f-1 "undo" each other: f(f-1(x)) = x and f-1(f(x)) = x
  • f-1 is NOT 1/f(x). It is the inverse function.
  • A function has an inverse only if it is one-to-one (passes the Horizontal Line Test)
  • To find f-1: replace f(x) with y, swap x and y, solve for y
  • Domain of f = range of f-1, and range of f = domain of f-1
  • Graphs of f and f-1 are reflections across the line y = x

Quadratic Functions

Standard Form vs. Vertex Form

  • Standard form: f(x) = ax² + bx + c
  • Vertex form: f(x) = a(x - h)² + k where (h, k) is the vertex
  • Convert standard to vertex by completing the square or using h = -b/(2a) and k = f(h)
  • a > 0: parabola opens up (vertex is a minimum)
  • a < 0: parabola opens down (vertex is a maximum)
  • |a| > 1: narrower than x². |a| < 1: wider than x².

Key Features of Parabolas

  • Vertex: (-b/(2a), f(-b/(2a)))
  • Axis of symmetry: x = -b/(2a) (vertical line through the vertex)
  • y-intercept: set x = 0, so y = c
  • x-intercepts (if they exist): solve ax² + bx + c = 0 using quadratic formula or factoring
  • The discriminant tells you the number of x-intercepts: Δ > 0 (two), Δ = 0 (one), Δ < 0 (none)

Applications of Quadratics

  • Maximum/minimum problems: the vertex gives the optimal value
  • Projectile motion: height = -16t² + v₀t + h₀ (in feet). Maximum height occurs at the vertex.
  • Revenue/profit optimization: find the vertex of the quadratic model
  • Area optimization: set up a quadratic and find the vertex

Polynomial Functions

Polynomial Basics

  • General form: f(x) = anxn + an-1xn-1 + ... + a1x + a0
  • The degree is the highest exponent. The leading coefficient is an.
  • A polynomial of degree n has at most n real zeros and at most n - 1 turning points
  • Domain of any polynomial is all real numbers (-∞, ∞)

End Behavior

DegreeLeading CoefficientLeft End (x → -∞)Right End (x → +∞)
EvenPositive↑ (up)↑ (up)
EvenNegative↓ (down)↓ (down)
OddPositive↓ (down)↑ (up)
OddNegative↑ (up)↓ (down)
Even degree: both ends go the same direction. Odd degree: ends go opposite directions. Positive leading coeff: right end goes up.

Zeros & Multiplicity

  • If (x - r) is a factor, then r is a zero of the polynomial
  • Multiplicity: the number of times a factor repeats
  • Odd multiplicity: the graph crosses the x-axis at that zero
  • Even multiplicity: the graph touches (bounces off) the x-axis at that zero
  • Rational Zero Theorem: possible rational zeros are ± (factors of constant term) / (factors of leading coefficient)

Division of Polynomials

  • Long division: works for dividing by any polynomial
  • Synthetic division: shortcut for dividing by (x - c) only
  • Remainder Theorem: if f(x) is divided by (x - c), the remainder is f(c)
  • Factor Theorem: (x - c) is a factor of f(x) if and only if f(c) = 0

Rational Functions

Rational Function Basics

  • Form: f(x) = p(x)/q(x) where p and q are polynomials and q(x) ≠ 0
  • Domain: all real numbers except where the denominator equals zero
  • Always check for common factors first (they create holes, not asymptotes)

Asymptotes

  • Vertical asymptote (VA): set the denominator = 0 (after canceling common factors). The graph approaches but never crosses a VA.
  • Horizontal asymptote (HA): compare the degrees of numerator and denominator:
  • Degree of top < bottom: HA: y = 0
  • Degree of top = bottom: HA: y = leading coeff of top / leading coeff of bottom
  • Degree of top > bottom: no HA (may have a slant/oblique asymptote)
  • Hole: occurs where a factor cancels from both numerator and denominator. Find the y-coordinate by evaluating the simplified function.

Graphing Rational Functions

  • Find the domain (exclude VA and holes)
  • Find x-intercepts: set numerator = 0
  • Find y-intercept: evaluate f(0)
  • Find VA, HA, and holes
  • Plot test points in each region between vertical asymptotes
  • The graph can cross a horizontal asymptote in the middle but approaches it at the ends

Exponential Functions

Exponential Function Basics

  • Form: f(x) = bx where b > 0 and b ≠ 1
  • Domain: (-∞, ∞). Range: (0, ∞). Always positive.
  • y-intercept: (0, 1) because b0 = 1
  • Horizontal asymptote: y = 0 (the x-axis)
  • b > 1: exponential growth (increasing)
  • 0 < b < 1: exponential decay (decreasing)
  • The function is one-to-one, so it has an inverse (logarithm)

The Number e

  • e ≈ 2.71828... (an irrational number)
  • Called the natural base
  • f(x) = ex is the natural exponential function
  • Arises naturally in continuous compound interest, population growth, and calculus

Compound Interest Formulas

TypeFormula
Compound n times/yearA = P(1 + r/n)nt
Continuous compoundingA = Pert

Where A = final amount, P = principal, r = annual rate (decimal), n = compounds per year, t = time in years.

Exponential Growth & Decay Models

  • General model: A(t) = A0ekt
  • k > 0: growth. k < 0: decay.
  • A0 = initial amount. t = time. A(t) = amount at time t.
  • Half-life: time for half the substance to decay. Set A = A0/2 and solve for t.
  • Doubling time: set A = 2A0 and solve for t.

Logarithmic Functions

Logarithm Definition

A logarithm is the inverse of an exponential:

y = logb(x) means by = x

  • "log base b of x equals y" means "b raised to the y equals x"
  • logb(bx) = x and blogb(x) = x (inverse properties)
  • Domain: (0, ∞). Range: (-∞, ∞). Vertical asymptote: x = 0.
  • Passes through (1, 0) because logb(1) = 0
  • Passes through (b, 1) because logb(b) = 1

Common & Natural Logarithms

  • log(x) = log10(x): common logarithm (base 10)
  • ln(x) = loge(x): natural logarithm (base e)
  • These are the two logs on your calculator
  • Change of base formula: logb(x) = ln(x)/ln(b) = log(x)/log(b)

Logarithm Properties

PropertyFormula
Product rulelogb(MN) = logb(M) + logb(N)
Quotient rulelogb(M/N) = logb(M) - logb(N)
Power rulelogb(Mp) = p · logb(M)
Log of 1logb(1) = 0
Log of the baselogb(b) = 1
Logs turn multiplication into addition, division into subtraction, and exponents into multiplication. They "bring exponents down."

Solving Exponential Equations

  • Same base method: if bm = bn, then m = n
  • Take the log of both sides: for equations like 3x = 20, take ln (or log) of both sides and use the power rule to bring x down
  • For equations with e: eexpression = k → expression = ln(k)

Solving Logarithmic Equations

  • Rewrite in exponential form: logb(expression) = k means bk = expression
  • If both sides are single logs with the same base: set the arguments equal
  • Use log properties to condense multiple logs into one before converting
  • Always check for extraneous solutions: the argument of a logarithm must be positive
