Exponent Rules

What Are Exponents?

An exponent tells you how many times to multiply a base by itself. In x3, the base is x and the exponent is 3, so x3 = x · x · x.

The Core Rules

There are a handful of rules that let you simplify expressions with exponents without expanding everything out. Here are the ones you need:

RuleWhat It SaysWhy It Works
Product rulexm · xn = xm+nYou're just combining all the x's being multiplied
Quotient rulexm / xn = xm-nCancel the x's that appear in both top and bottom
Power rule(xm)n = xmnYou're raising a power to a power, so multiply them
Zero exponentx0 = 1Follows from the quotient rule: xn/xn = x0 = 1
Negative exponentx-n = 1/xnA negative exponent means "take the reciprocal"
Worked Example

Simplify: x4 · x3

1.Same base (x), so add the exponents: 4 + 3 = 7
2.Answer: x7
Worked Example

Simplify: (x2)5

1.Power to a power, so multiply: 2 × 5 = 10
2.Answer: x10
Any nonzero number raised to the 0 power is 1. So 50 = 1, (-3)0 = 1, and (1000)0 = 1.

Factoring

What Is Factoring?

Factoring means rewriting an expression as a product of simpler pieces. It's like "un-distributing." If you know that 3 × 5 = 15, then factoring 15 gives you 3 × 5. In algebra we do the same thing with expressions involving x.

Always start by pulling out the GCF (greatest common factor) before trying anything else.

Difference of Squares

This is the pattern you'll see most often. If you have something squared minus something else squared, it factors neatly:

a2 - b2 = (a + b)(a - b)

Worked Example

Factor: x2 - 25

1.Recognize the pattern: x2 is a perfect square, and 25 = 52.
2.Apply the formula with a = x and b = 5.
3.Answer: (x + 5)(x - 5)
This only works for subtraction. There is no "sum of squares" factoring formula for real numbers. x2 + 25 does not factor.

Factoring Trinomials (ax2 + bx + c)

When a = 1 (like x2 + 7x + 12), you need two numbers that multiply to c and add to b.

Worked Example

Factor: x2 + 7x + 12

1.Need two numbers that multiply to 12 and add to 7.
2.Try factor pairs of 12: (1,12), (2,6), (3,4).
3.3 + 4 = 7. That's the one.
4.Answer: (x + 3)(x + 4)

Solving Equations

Linear Equations

A linear equation has x to the first power (no x2 or higher). The goal is to get x by itself on one side. You do this by "undoing" what's been done to x using inverse operations: undo addition with subtraction, undo multiplication with division.

Worked Example

Solve: 3x + 7 = 22

1.Subtract 7 from both sides: 3x = 15
2.Divide both sides by 3: x = 5
3.Check: 3(5) + 7 = 15 + 7 = 22 ✓

Quadratic Equations

A quadratic has x2 as its highest power. Standard form is ax2 + bx + c = 0. There are several ways to solve these:

Method 1: Factoring. If the expression factors, set each factor equal to zero.

Method 2: Quadratic formula. This always works:

x = (-b ± √(b2 - 4ac)) / 2a

Worked Example — Factoring

Solve: x2 - 5x + 6 = 0

1.Find two numbers that multiply to 6 and add to -5. That's -2 and -3.
2.Factor: (x - 2)(x - 3) = 0
3.Set each factor to 0: x - 2 = 0 gives x = 2. x - 3 = 0 gives x = 3.
4.Solutions: x = 2 and x = 3
Worked Example — Quadratic Formula

Solve: 2x2 + 3x - 5 = 0

1.Identify a = 2, b = 3, c = -5.
2.Discriminant: b2 - 4ac = 9 - 4(2)(-5) = 9 + 40 = 49.
3.√49 = 7, so x = (-3 ± 7) / (2 · 2) = (-3 ± 7) / 4.
4.x = (-3 + 7)/4 = 4/4 = 1, or x = (-3 - 7)/4 = -10/4 = -5/2.
5.Solutions: x = 1 and x = -5/2

The Discriminant: How Many Solutions?

The expression under the square root in the quadratic formula, b2 - 4ac, is called the discriminant. It tells you what kind of solutions you'll get before you even solve:

If b2 - 4ac is...Then you get...
PositiveTwo different real solutions
ZeroExactly one real solution (a "double root")
NegativeNo real solutions
A negative discriminant means you'd be taking the square root of a negative number, which has no answer in the real number system.

Absolute Value Equations

The absolute value of a number is its distance from zero, so it's always positive or zero. To solve |expression| = k, you write two equations: expression = k, and expression = -k.

Worked Example

Solve: |x - 3| = 7

1.Write two equations: x - 3 = 7 and x - 3 = -7
2.Solve: x = 10 and x = -4
3.Check: |10 - 3| = |7| = 7 ✓ and |-4 - 3| = |-7| = 7 ✓

Inequalities

The One Big Rule

You solve inequalities almost exactly like equations, with one critical exception: when you multiply or divide both sides by a negative number, you must flip the inequality sign.

Worked Example

Solve: -2x + 5 > 11

1.Subtract 5 from both sides: -2x > 6
2.Divide by -2 and flip the sign: x < -3
3.In interval notation: (-∞, -3)

Interval Notation

Interval notation is a compact way to write the solution set. Parentheses mean "not included" (open circle), brackets mean "included" (closed circle). Infinity always gets a parenthesis because you can never reach it.

MeaningNotation
x is between 2 and 7, not including endpoints(2, 7)
x is between 2 and 7, including both[2, 7]
x is greater than 5(5, ∞)
x is less than or equal to 3(-∞, 3]

Absolute Value Inequalities

These follow a simple pattern. Think of absolute value as distance from zero:

|x| < a means x is between -a and a: -a < x < a

|x| > a means x is outside that range: x < -a or x > a

Worked Example

Solve: |2x + 1| < 5

1."Less than" means between: -5 < 2x + 1 < 5
2.Subtract 1 from all three parts: -6 < 2x < 4
3.Divide all parts by 2: -3 < x < 2
4.In interval notation: (-3, 2)
Memory trick: "Less thAND" (the solution is one interval joined by AND). "GreatOR" (two separate pieces joined by OR).

Functions

What Is a Function?

A function is a rule that takes each input (x) and produces exactly one output (y). Think of it as a machine: you put in a number, and one number comes out. If an input could give two different outputs, it's not a function.

Vertical Line Test: If you can draw a vertical line anywhere on a graph and it hits the curve more than once, the graph is not a function.

Notation: f(x) means "the output of function f when the input is x." It is NOT f times x.

Evaluating a Function

To evaluate f(a), replace every x in the formula with a.

Worked Example

If f(x) = 2x2 - 3x + 1, find f(4).

1.Replace every x with 4: f(4) = 2(4)2 - 3(4) + 1
2.= 2(16) - 12 + 1
3.= 32 - 12 + 1 = 21

Domain: What Can You Plug In?

The domain is the set of all x-values you're allowed to use. Most of the time, the domain is all real numbers. But there are three situations that create restrictions:

1. Fractions: The denominator cannot be zero. Find what makes it zero and exclude that.

2. Square roots (even roots): The expression under the root must be ≥ 0.

3. Logarithms: The argument must be > 0.

Worked Example

Find the domain of f(x) = √(x - 3)

1.The expression under the root must be ≥ 0: x - 3 ≥ 0
2.Solve: x ≥ 3
3.Domain in interval notation: [3, ∞)
Worked Example

Find the domain of g(x) = 1/(x2 - 9)

1.Set the denominator ≠ 0: x2 - 9 ≠ 0
2.Factor: (x + 3)(x - 3) ≠ 0, so x ≠ 3 and x ≠ -3
3.Domain: all real numbers except x = 3 and x = -3

Graph Transformations

How Graphs Move

Once you know the shape of a basic graph (like x2 or √x), you can shift, stretch, or flip it using simple rules. Starting from a graph y = f(x):

ChangeEffect on the Graph
f(x) + kShift up k units
f(x) - kShift down k units
f(x - h)Shift right h units
f(x + h)Shift left h units
-f(x)Flip over the x-axis
f(-x)Flip over the y-axis
Worked Example

Describe the graph of y = (x - 2)2 + 5

1.Start with the parent function y = x2 (a U-shaped parabola at the origin).
2.(x - 2) means shift right 2 (opposite the sign inside).
3.+ 5 on the outside means shift up 5.
4.Result: a parabola with its vertex at (2, 5).
Horizontal shifts feel backwards: f(x - 3) shifts RIGHT 3, and f(x + 3) shifts LEFT 3. Think "opposite of the sign inside."

Inverse Functions

What Is an Inverse?

An inverse function undoes what the original function does. If f turns 2 into 7, then f-1 turns 7 back into 2. The notation f-1 does NOT mean 1/f(x). It means the inverse function.

A function can only have an inverse if it's one-to-one, meaning each y-value comes from only one x-value. You check this with the Horizontal Line Test: if any horizontal line crosses the graph more than once, it fails.

How to Find an Inverse

Three steps: (1) replace f(x) with y, (2) swap x and y, (3) solve for y.

Worked Example

Find the inverse of f(x) = 3x - 5

1.Write y = 3x - 5
2.Swap x and y: x = 3y - 5
3.Solve for y: x + 5 = 3y, so y = (x + 5)/3
4.Answer: f-1(x) = (x + 5)/3
The graphs of f and f-1 are mirror images across the line y = x.

Quadratic Functions & Parabolas

The Vertex: The Most Important Point

Every parabola y = ax2 + bx + c has a highest or lowest point called the vertex. You find it with this formula:

x = -b / (2a), then plug that x back in to find y.

If a > 0, the parabola opens up (the vertex is a minimum). If a < 0, it opens down (the vertex is a maximum).

Worked Example

Find the vertex of f(x) = -2x2 + 8x - 3

1.a = -2, b = 8. So x = -8 / (2 · -2) = -8 / -4 = 2
2.f(2) = -2(4) + 8(2) - 3 = -8 + 16 - 3 = 5
3.Vertex: (2, 5). Since a = -2 < 0, this is a maximum.

Polynomial Basics

End Behavior

The end behavior of a polynomial is what happens to the graph as x goes very far left or very far right. It depends on two things: the degree (highest exponent) and the leading coefficient (the number in front of the highest power).

DegreeLeading Coeff.Left EndRight End
EvenPositive↑ up↑ up
EvenNegative↓ down↓ down
OddPositive↓ down↑ up
OddNegative↑ up↓ down
Even degree = both ends go the same way. Odd degree = ends go opposite ways.

Zeros and Multiplicity

When a factor like (x - 2) appears in a polynomial, x = 2 is a zero (the graph hits the x-axis there). The multiplicity is how many times that factor repeats:

Odd multiplicity (1, 3, 5...): the graph crosses the x-axis at that zero.

Even multiplicity (2, 4, 6...): the graph touches and bounces off the x-axis.

Worked Example

Describe the behavior at each zero of f(x) = (x + 1)2(x - 3)

1.Zero at x = -1 with multiplicity 2 (even): graph bounces off the axis.
2.Zero at x = 3 with multiplicity 1 (odd): graph crosses the axis.

Rational Functions

Asymptotes: Lines the Graph Approaches But Never Reaches

A rational function is a fraction where the top and bottom are both polynomials: f(x) = p(x)/q(x).

Vertical asymptote (VA): Set the denominator equal to zero and solve. The graph shoots toward ±∞ near these x-values.

Horizontal asymptote (HA): Compare the degree of the numerator (top) to the degree of the denominator (bottom):

If...Then the HA is...
Degree of top < degree of bottomy = 0
Degree of top = degree of bottomy = (leading coeff. of top) / (leading coeff. of bottom)
Degree of top > degree of bottomNo horizontal asymptote
Worked Example

Find the asymptotes of f(x) = (3x + 1) / (x - 4)

1.VA: set x - 4 = 0, so VA at x = 4.
2.HA: degree of top = 1, degree of bottom = 1 (equal).
3.HA = 3/1 = 3, so HA at y = 3.

Exponentials & Logarithms

Exponential Functions

An exponential function has the variable in the exponent: f(x) = bx, where b > 0 and b ≠ 1. The graph always passes through (0, 1) because anything to the zero power is 1.

If b > 1, the function grows (gets bigger fast). If 0 < b < 1, the function decays (shrinks toward zero).

The special base e ≈ 2.71828 shows up constantly in science and finance.

What Is a Logarithm?

A logarithm answers the question: "What exponent do I need?"

logb(x) = y means by = x

In plain English: "log base b of x" asks, "what power do I raise b to in order to get x?"

Worked Example

Evaluate: log2(32)

1.Ask: "2 to what power gives 32?"
2.25 = 32
3.Answer: log2(32) = 5

Two key facts: logb(1) = 0 (because b0 = 1) and logb(b) = 1 (because b1 = b).

Log Properties

Logarithms have three properties that let you break apart or combine expressions. These mirror the exponent rules:

PropertyFormulaWhat It Does
Product rulelog(MN) = log(M) + log(N)Turns multiplication into addition
Quotient rulelog(M/N) = log(M) - log(N)Turns division into subtraction
Power rulelog(Mp) = p · log(M)Brings the exponent down as a multiplier

Solving Exponential Equations

When the variable is in the exponent and you can't match bases, take the log of both sides and use the power rule to bring the exponent down.

Worked Example

Solve: 5x = 200

1.Take ln of both sides: ln(5x) = ln(200)
2.Power rule brings x down: x · ln(5) = ln(200)
3.Divide: x = ln(200) / ln(5) ≈ 3.29

Compound Interest

Two formulas to know:

Compounded n times per year: A = P(1 + r/n)nt

Compounded continuously: A = Pert

Where A = final amount, P = principal (starting amount), r = annual rate as a decimal, n = times compounded per year, t = years.

Worked Example

$5,000 is invested at 4% compounded continuously. How much after 10 years?

1.Use A = Pert with P = 5000, r = 0.04, t = 10.
2.A = 5000 · e0.4 ≈ 5000 · 1.4918 ≈ $7,459.12
